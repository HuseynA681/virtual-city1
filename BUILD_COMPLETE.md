## 💡 Quick Tips

### If Something Doesn't Work
1. Check `SETUP.md` for troubleshooting
2. Ensure MongoDB is running
3. Check that ports 5000 and 3000 are free
4. Clear browser cache (Ctrl+Shift+Delete)
5. Restart both servers

### To Understand the Code
1. Start with `backend/server.js`
2. Then check `backend/models/`
3. See how routes use models
4. Check `frontend/src/App.js` for routing
5. Open DevTools to see network requests

### To Add Features
1. Create database schema in `models/`
2. Add API route in `routes/`
3. Create frontend component in `components/`
4. Connect with axios calls
5. Test in browser
